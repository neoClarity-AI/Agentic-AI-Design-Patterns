# Agentic AI Design Patterns

This repo is inspired by the "Gang of Four" [Design Patterns](https://en.wikipedia.org/wiki/Design_Patterns) book (1994), which had major impact on object-oriented software development. This book provided value to the software development community in two ways: it created a shared vocabulary and documented proven, reusable solutions for recurring design problems. The objective of **Agentic AI Design Patterns** is to provide similar value to the development of Agentic AI systems.

Access the document here: [Agentic AI Design Patterns](Agentic%20AI%20Design%20Patterns.md)

The reference implementation contains many, but not all, of the design patterns described in this document.
